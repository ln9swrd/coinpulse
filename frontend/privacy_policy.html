🔒 개인정보처리방침

시행일: 2025년 1월 1일

코인펄스(CoinPulse)는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수하고 있습니다. 본 방침은 회사가 수집하는 개인정보의 항목, 이용 목적, 보유 기간 등을 안내합니다.

제1조 (수집하는 개인정보의 항목 및 수집 방법)

1.1 필수 수집 항목

수집 시점 수집 항목 수집 목적
회원 가입 이메일, 비밀번호, 닉네임 회원 식별, 서비스 제공
자동매매 이용 업비트 API 키 (Access Key, Secret Key) 자동매매 실행
텔레그램 연동 텔레그램 Chat ID 알림 발송
결제 결제 정보 (카드 번호 일부, 결제 일시) 구독료 결제

1.2 선택 수집 항목

  • 프로필 사진
  • 추천인 코드

1.3 자동 수집 항목

서비스 이용 과정에서 다음 정보가 자동으로 생성되어 수집됩니다:

  • IP 주소
  • 쿠키 (Cookie)
  • 접속 로그 (접속 시간, 이용 기록)
  • 기기 정보 (OS, 브라우저 종류)

1.4 수집 방법

  • 회원 가입 시 웹사이트를 통한 직접 입력
  • API 키 등록 페이지를 통한 직접 입력
  • 로그 분석 도구를 통한 자동 수집

제2조 (개인정보의 이용 목적)

2.1 회원 관리

  • 회원 식별 및 본인 확인
  • 부정 이용 방지 및 비인가 사용 방지
  • 회원 탈퇴 의사 확인
  • 고지사항 전달

2.2 서비스 제공

  • 급등 예측 정보 제공
  • 자동매매 실행 (업비트 API 키 사용)
  • 텔레그램 알림 발송
  • 포트폴리오 관리

2.3 결제 및 환불

  • 구독료 결제 처리
  • 환불 요청 처리
  • 결제 내역 관리

2.4 서비스 개선

  • 신규 서비스 개발
  • 통계 분석 및 서비스 품질 향상
  • 맞춤형 서비스 제공

제3조 (개인정보의 보유 및 이용 기간)

3.1 원칙

회사는 개인정보 수집 및 이용 목적이 달성된 후에는 해당 정보를 지체 없이 파기합니다. 단, 관련 법령에 따라 보존해야 하는 경우 일정 기간 보관합니다.

3.2 보유 기간

항목 보유 기간 근거
회원 정보 회원 탈퇴 시까지 서비스 이용 계약
API 키 회원 탈퇴 또는 삭제 요청 시까지 자동매매 서비스 제공
결제 정보 5년 전자상거래법

거래 기록 5년 전자금융거래법

3.3 법령에 따른 보관

관련 법령에 따라 다음 정보를 일정 기간 보관합니다:

  • 소비자 불만 기록: 3년 (전자상거래법)
  • 대금 결제 기록: 5년 (전자상거래법)
  • 계약 또는 청약철회 기록: 5년 (전자상거래법)
  • 로그인 기록: 3개월 (통신비밀보호법)

제5조 (개인정보의 처리 위탁)

5.1 위탁 업무

회사는 서비스 제공을 위해 다음과 같이 개인정보 처리를 위탁합니다:

수탁업체 위탁 업무
AWS (Amazon Web Services) 서버 호스팅, 데이터 저장
토스페이먼츠 결제 대행

5.2 위탁 관리

  • 수탁업체와 개인정보 보호 계약을 체결합니다
  • 수탁업체가 개인정보를 안전하게 처리하도록 관리·감독합니다
  • 위탁 내용 변경 시 본 방침을 통해 공지합니다

제6조 (이용자의 권리와 행사 방법)

6.1 이용자의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 열람 요구권: 자신의 개인정보를 조회하고 확인할 권리
  • 정정 요구권: 잘못된 개인정보를 수정할 권리
  • 삭제 요구권: 개인정보 삭제를 요청할 권리
  • 처리 정지 요구권: 개인정보 처리를 일시 중지할 권리

6.2 권리 행사 방법

  • 웹사이트 내 '마이페이지' → '개인정보 관리'
  • 이메일: privacy@coinpulse.com (가상 이메일)
  • 고객센터: 텔레그램 문의

제7조 (개인정보의 파기)

7.1 파기 절차

  1. 보유 기간이 만료되거나 처리 목적이 달성된 개인정보를 선정
  2. 개인정보 보호책임자의 승인 획득
  3. 개인정보 파기

7.2 파기 방법

  • 전자파일: 복구 불가능한 방법으로 영구 삭제
  • 종이문서: 파쇄기로 분쇄 또는 소각

제8조 (개인정보의 안전성 확보 조치)

8.1 기술적 조치

  • 개인정보 암호화 (AES-256)
  • API 키 암호화 저장
  • 해킹 방지를 위한 방화벽 설치
  • 백신 프로그램 설치 및 주기적 점검
  • SSL 인증서를 통한 암호화 통신

8.2 관리적 조치

  • 개인정보 처리 직원의 최소화 및 교육
  • 개인정보 접근 권한 관리
  • 접속 기록 보관 및 위변조 방지
  • 정기적인 보안 점검

8.3 물리적 조치

  • 전산실, 자료 보관실 등의 출입 통제
  • 클라우드 서버 보안 (AWS)

제9조 (쿠키의 운영 및 거부)

9.1 쿠키란?

쿠키(Cookie)는 웹사이트가 이용자의 브라우저에 전송하는 소량의 정보로, 이용자의 컴퓨터 하드디스크에 저장됩니다.

9.2 쿠키 사용 목적

  • 로그인 상태 유지
  • 이용자 맞춤형 서비스 제공
  • 접속 빈도 분석

9.3 쿠키 거부 방법

  • Chrome: 설정 → 개인정보 및 보안 → 쿠키 및 기타 사이트 데이터
  • Edge: 설정 → 쿠키 및 사이트 권한
  • Firefox: 설정 → 개인정보 및 보안 → 쿠키 및 사이트 데이터

※ 쿠키 저장을 거부할 경우 일부 서비스 이용에 제한이 있을 수 있습니다.
