🚀 시작하기

코인펄스는 기술적 분석 기반 암호화폐 자동매매 보조 플랫폼입니다. 3단계만으로 시작할 수 있습니다.

Step 1: 회원가입

  1. 회원가입 페이지에 접속합니다
  2. 이메일 주소와 비밀번호를 입력합니다
  3. 이용약관과 개인정보처리방침에 동의합니다
  4. 7일 무료 체험이 자동으로 시작됩니다
💡 무료 체험 혜택:
  • ✓ 1개 코인 모니터링
  • ✓ 하루 3회 자동매매
  • ✓ 기본 기술적 지표
  • ✓ 신용카드 등록 불필요

🔑 업비트 API 키 설정

자동매매를 위해서는 업비트 API 키가 필요합니다.

1. 업비트에서 API 키 발급받기

  1. 업비트 웹사이트(upbit.com)에 로그인
  2. 마이페이지 → 고객센터 → Open API 관리 메뉴 이동
  3. 'Open API 키 발급' 버튼 클릭
  4. 다음 권한만 선택:
    • ✓ 자산 조회 (필수)
    • ✓ 주문 조회 (필수)
    • ✓ 주문하기 (필수)
    • ✗ 출금하기 (선택하지 않음)
  5. IP 주소 제한 설정 (선택사항, 권장)
  6. Access Key와 Secret Key 복사 및 안전하게 보관
⚠️ 중요 보안 사항:
  • 출금 권한은 절대 활성화하지 마세요
  • API 키는 타인과 공유하지 마세요
  • 주기적으로 API 키를 재발급하는 것을 권장합니다

2. 코인펄스에 API 키 등록하기

  1. 대시보드에서 '설정' 메뉴 클릭
  2. 'API 키 관리' 섹션으로 이동
  3. Access Key와 Secret Key 입력
  4. '연결 테스트' 버튼으로 정상 작동 확인
  5. '저장' 버튼 클릭
🔒 보안 정책: API 키는 AES-256 암호화되어 저장되며, 서버에서도 복호화된 상태로 메모리에 보관되지 않습니다.

📊 대시보드 사용법

대시보드는 전체 포트폴리오 현황을 한눈에 파악할 수 있는 메인 화면입니다.

주요 기능

  • 총 자산 현황: KRW 기준 총 보유 자산 표시
  • 코인별 보유 현황: 각 코인의 보유량, 평균 매수가, 수익률
  • 최근 거래 내역: 최근 10개 거래 히스토리
  • 실시간 가격: 주요 코인의 실시간 시세
  • 자동매매 상태: 현재 활성화된 자동매매 전략

네비게이션

  • Swing Trading: 스윙 트레이딩 관리 페이지
  • Real-time Monitor: 실시간 모니터링 대시보드
  • Surge Prediction: 급등 예측 분석 페이지
  • Trading Chart: 고급 차트 분석 도구

🤖 자동매매 설정

기술적 지표를 활용한 자동매매 전략을 설정할 수 있습니다.

지원하는 기술적 지표

  • SMA (단순이동평균): MA20, MA60, MA200, MA300
  • RSI (상대강도지수): 과매수/과매도 판단
  • MACD: 추세 전환 감지
  • 볼린저밴드: 변동성 분석
  • SuperTrend: 추세 방향 확인

전략 설정 방법

  1. 대시보드에서 '자동매매 설정' 클릭
  2. 거래할 코인 선택 (무료: 1개, 프리미엄: 3개, 프로: 5개)
  3. 매수/매도 조건 설정:
    • 매수: RSI < 30 AND 가격 < 볼린저밴드 하단
    • 매도: RSI > 70 OR 수익률 > 5%
  4. 거래 금액 및 횟수 제한 설정
  5. '활성화' 버튼 클릭
⚠️ 리스크 관리:
  • 자동매매는 보조 도구이며, 수익을 보장하지 않습니다
  • 손절매 조건을 반드시 설정하세요 (예: -5% 손실 시 매도)
  • 일일 최대 거래 금액을 제한하세요
  • 정기적으로 전략 성과를 모니터링하세요

📈 스윙 트레이딩

중장기 관점에서 수익을 추구하는 스윙 트레이딩 기능입니다.

주요 기능

  • 포지션 관리: 현재 보유 중인 스윙 포지션 현황
  • 진입 타이밍: 기술적 지표 기반 매수 시그널
  • 목표가 설정: 자동 익절 목표가 설정
  • 손절 관리: 자동 손절 조건 설정
  • 거래 히스토리: 과거 스윙 거래 내역 및 수익률

스윙 트레이딩 전략

추천 전략:
  • 진입: MA20 골든크로스 + RSI 40~60
  • 익절: +10% 수익 또는 MA60 데드크로스
  • 손절: -5% 손실 또는 볼린저밴드 하단 이탈
  • 보유 기간: 3~14일

⚡ 급등 예측

AI 기반 급등 예측 시스템으로 24시간 내 급등 가능성이 높은 코인을 분석합니다.

예측 정확도

📊 백테스트 결과 (2024년 데이터):
  • 전체 정확도: 81.25%
  • 급등 감지율: 75%
  • 평균 예측 시간: 4~6시간 선행

활용 방법

  • 급등 후보 코인을 실시간으로 확인
  • 각 코인의 급등 확률 및 근거 분석
  • 텔레그램 알림으로 급등 시그널 수신 (프로 플랜)
  • 백테스트 결과로 과거 성과 검증
⚠️ 주의사항: 급등 예측은 참고 자료이며, 투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다.

🔒 보안 가이드

계정 보안

  • 강력한 비밀번호 사용 (8자 이상, 영문+숫자+특수문자)
  • 정기적인 비밀번호 변경 (3개월마다 권장)
  • 타인과 계정 정보 공유 금지

API 키 보안

  • 출금 권한은 절대 활성화하지 마세요
  • 주기적으로 API 키 재발급 (6개월마다 권장)
  • 의심스러운 거래 발견 시 즉시 API 키 삭제

시스템 보안

코인펄스 보안 정책:
  • ✓ AES-256 암호화 (API 키, 비밀번호)
  • ✓ HTTPS 통신 (모든 데이터 전송)
  • ✓ 접속 로그 관리 (1년 보관)
  • ✓ 정기 보안 점검 (월 1회)

🔧 문제 해결

API 연결 오류

증상: "API 연결 실패" 오류 메시지

해결 방법:

  1. 업비트에서 API 키가 정상적으로 발급되었는지 확인
  2. 필수 권한(자산조회, 주문조회, 주문하기)이 활성화되어 있는지 확인
  3. IP 제한이 설정된 경우 현재 IP가 허용 목록에 있는지 확인
  4. API 키를 재발급하여 다시 시도

자동매매가 실행되지 않음

증상: 자동매매가 활성화되어 있으나 거래가 실행되지 않음

해결 방법:

  1. 대시보드에서 자동매매 상태가 '활성'인지 확인
  2. 설정한 매수/매도 조건이 현재 시장 상황에 맞는지 확인
  3. KRW 잔고가 충분한지 확인
  4. 일일 거래 한도에 도달하지 않았는지 확인

로그인 문제

증상: 로그인이 되지 않거나 세션이 자주 끊김

해결 방법:

  1. 브라우저 쿠키 설정이 활성화되어 있는지 확인
  2. 캐시 및 쿠키 삭제 후 재시도
  3. 다른 브라우저에서 시도
  4. 비밀번호 재설정 이용
