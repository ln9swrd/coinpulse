⚠️ 필수 권한 설정

업비트에서 API 키를 발급할 때 다음 권한을 반드시 활성화해주세요:

  • ✅ 자산 조회 - 포트폴리오 확인용
  • ✅ 주문 조회 - 거래 내역 조회용
  • ✅ 주문 하기 - 자동매매 실행용 (선택)

※ 자동매매를 사용하지 않는다면 "주문 하기" 권한은 비활성화하는 것을 권장합니다.

📘 업비트 API 키 발급 방법

  1. 업비트 Open API 관리 페이지 에 접속합니다.
  2. 로그인 후 "Open API 사용하기" 버튼을 클릭합니다.
  3. 본인 인증 (OTP 또는 SMS)을 완료합니다.
  4. 필수 권한 선택:
    • ✅ 자산 조회
    • ✅ 주문 조회
    • ✅ 주문 하기 (자동매매 사용 시)
  5. IP 주소 제한은 "사용 안 함"으로 설정합니다.
  6. 발급된 Access Key와 Secret Key를 복사하여 위 입력란에 붙여넣습니다.
  7. Secret Key는 한 번만 표시되므로 반드시 안전한 곳에 저장하세요!

💡 팁: API 키는 절대 다른 사람과 공유하지 마세요. 만약 유출되었다면 즉시 업비트에서 해당 키를 삭제하고 새로 발급받으세요.
