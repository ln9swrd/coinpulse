Telegram 연동하기

------
15분 내에 입력해주세요

연동 방법

  1. Telegram에서 @coinpulse_surge_sinsi_bot 검색
  2. /start 명령어로 봇 시작
  3. /link [코드] 명령어 입력 (예: /link 123456)
  4. 연동 완료 메시지 확인
알림 설정: 연동 후 자동으로 트레이딩 시그널 알림을 받게 됩니다. 알림을 중지하려면 Telegram 봇에서 /stop 명령어를 사용하세요.

연동 해제

Telegram 연동을 해제하면 더 이상 트레이딩 시그널 알림을 받지 않습니다.
