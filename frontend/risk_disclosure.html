중요 안내

본 투자위험고지서는 가상자산 자동매매 서비스를 이용하시기 전에 반드시 읽고 이해하셔야 합니다. 서비스 이용은 본 고지서의 내용을 충분히 이해하고 동의한 것으로 간주됩니다.

1. 가상자산 투자의 일반적 위험

1.1 원금 손실 위험

가상자산 투자는 원금 전액 손실 가능성이 있는 고위험 투자입니다. 가상자산 시장은 주식, 채권 등 전통 금융상품보다 변동성이 매우 높으며, 단기간에 큰 손실이 발생할 수 있습니다.

1.2 가격 변동성 위험

  • 급격한 가격 변동: 가상자산 가격은 하루에 ±30% 이상 변동할 수 있습니다
  • 시장 조작 위험: 소수의 대량 보유자(whale)에 의한 시장 조작 가능성
  • 유동성 위험: 거래량이 부족한 코인의 경우 원하는 가격에 매도하지 못할 수 있습니다
  • 기술적 리스크: 블록체인 기술 결함, 하드포크, 해킹 등의 위험

1.3 법적·규제 위험

가상자산은 법정화폐가 아니며, 정부나 중앙은행의 보증이 없습니다. 향후 정부 규제 강화, 거래소 폐쇄, 거래 금지 등으로 인해 투자금을 회수하지 못할 수 있습니다.

⛔ 절대 금지 사항

  • 생활비, 학자금, 빚으로 투자하지 마십시오
  • 대출을 받아 투자하지 마십시오
  • 감당할 수 없는 금액을 투자하지 마십시오
  • 투자 결정을 타인에게 의존하지 마십시오

2. 자동매매 시스템의 특정 위험

2.1 알고리즘 예측 한계

본 서비스의 급등 예측 알고리즘은 과거 데이터를 기반으로 개발되었으며, 미래 수익을 보장하지 않습니다. 백테스트 결과와 실제 거래 결과는 다를 수 있습니다.

  • 백테스트 승률: 과거 데이터 기준 45-50% (절반은 손실)
  • 평균 수익률: 1-2% 수준 (수수료 제외 시)
  • 슬리피지: 실제 체결가는 목표가와 다를 수 있습니다

2.2 시스템 오류 위험

  • 서버 장애: 시스템 다운 시 자동매매가 중단될 수 있습니다
  • API 오류: 업비트 API 장애 시 거래가 실패할 수 있습니다
  • 네트워크 지연: 통신 지연으로 원하는 가격에 매매하지 못할 수 있습니다
  • 버그: 소프트웨어 버그로 인한 예기치 않은 거래 발생 가능성

🔒 당사의 데이터 사용 및 한계

데이터 출처:

  • 모든 시세 데이터는 업비트(Upbit) 공식 API를 통해 실시간으로 조회됩니다
  • 당사는 자체 가격 데이터를 생성하거나 조작하지 않습니다
  • 가상자산 가격이나 거래량에 영향을 줄 수 있는 능력이 없습니다

당사가 할 수 없는 것:

  • ❌ 가격 조작
  • ❌ 거래량 조작
  • ❌ 호가 조작
  • ❌ 시장 정보 선점

2.3 거래 수수료 및 세금

  • 거래 수수료: 업비트 매매 수수료 0.05% (매수/매도 각각)
  • 세금: 가상자산 소득세는 사용자 책임 (향후 과세 가능성)
  • 출금 수수료: 업비트 출금 시 별도 수수료 발생

3. 투자 결정 및 책임

3.1 사용자 책임

모든 투자 결정 및 그로 인한 손익은 전적으로 사용자 본인의 책임입니다. 당사는 다음 사항에 대해 책임지지 않습니다:

  • 투자 손실 (원금 손실 포함)
  • 예상 수익을 얻지 못한 경우
  • 시스템 오류로 인한 손실
  • 업비트 거래소 장애로 인한 손실
  • 규제 변경으로 인한 손실

3.2 서비스 성격

본 서비스는 투자 보조 도구일 뿐이며:

  • ❌ 투자자문업이 아닙니다
  • ❌ 투자일임업이 아닙니다
  • ❌ 투자 권유가 아닙니다
  • ❌ 매매 추천이 아닙니다
  • ✅ 정보 제공 목적의 참고 도구입니다

3.3 수익 보장 불가

당사는 어떠한 수익도 보장하지 않으며, 다음을 명시합니다:

  • 과거 백테스트 성과는 미래 수익을 보장하지 않습니다
  • 승률 45-50%는 절반은 손실이라는 의미입니다
  • 평균 수익률 1-2%는 수수료 제외 시 수익이 더 낮을 수 있습니다
  • 시장 상황에 따라 연속 손실이 발생할 수 있습니다

4. 개인정보 및 API 키 관리

4.1 API 키 보안

  • 업비트 API 키는 사용자가 직접 생성하고 관리합니다
  • API 키는 암호화되어 저장되지만, 해킹 위험은 항상 존재합니다
  • API 키 유출 시 자산 손실 위험이 있으므로 각별히 주의하십시오
  • 절대 다른 사람에게 API 키를 공유하지 마십시오

4.2 개인정보 처리

당사는 서비스 제공을 위해 최소한의 개인정보를 수집하며, 자세한 내용은 개인정보처리방침을 참조하십시오.

5. 서비스 중단 및 변경

5.1 서비스 중단 가능성

다음의 경우 사전 통보 없이 서비스가 중단될 수 있습니다:

  • 업비트 API 정책 변경
  • 정부 규제 강화
  • 시스템 유지보수
  • 불가항력 (천재지변, 전쟁, 해킹 등)

5.2 서비스 변경

당사는 서비스 개선을 위해 알고리즘, 수수료, 기능 등을 변경할 수 있으며, 중요한 변경사항은 사전 공지됩니다.

✅ 투자위험 확인

본 투자위험고지서를 충분히 읽고 이해했으며, 다음 사항을 확인합니다:

  1. 가상자산 투자는 원금 손실 위험이 있으며, 투자금 전액을 잃을 수 있음을 이해했습니다
  2. 과거 성과가 미래 수익을 보장하지 않음을 이해했습니다
  3. 당사는 업비트 공개 API만 사용하며, 가격 조작이 불가능함을 확인했습니다
  4. 투자 손실 책임은 전적으로 본인에게 있음을 동의합니다
  5. 자동매매는 보조 도구일 뿐이며, 최종 투자 결정은 본인이 합니다
  6. 감당할 수 있는 금액만 투자하겠습니다
