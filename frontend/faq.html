자주 묻는 질문

코인펄스 사용 중 궁금한 점을 찾아보세요

Q. 회원가입은 어떻게 하나요?

+

회원가입은 매우 간단합니다:

  1. 회원가입 페이지에 접속
  2. 이메일 주소와 비밀번호 입력
  3. 이용약관 동의
  4. 7일 무료 체험 자동 시작
💡 팁: 무료 체험 기간 중 신용카드 등록이 필요하지 않습니다.

Q. 비밀번호를 잊어버렸어요

+

비밀번호를 잊어버린 경우:

  1. 로그인 페이지에서 '비밀번호 찾기' 클릭
  2. 가입 시 사용한 이메일 주소 입력
  3. 이메일로 전송된 비밀번호 재설정 링크 클릭
  4. 새 비밀번호 설정

Q. 회원 탈퇴는 어떻게 하나요?

+

대시보드 → 설정 → 계정 관리 → 회원 탈퇴 메뉴에서 탈퇴할 수 있습니다.

⚠️ 주의: 탈퇴 시 모든 데이터가 삭제되며 복구할 수 없습니다. 유료 구독은 미리 취소해 주세요.

Q. 요금제는 어떻게 되나요?

+

코인펄스는 4가지 요금제를 제공합니다:

  • 무료 체험 (₩0): 7일간 1개 코인, 하루 3회 거래
  • 베이직 (₩19,900/월): 2개 코인, 무제한 거래
  • 프리미엄 (₩39,900/월): 3개 코인, 고급 지표, 우선 지원
  • 프로 (₩99,900/월): 5개 코인, VIP 전략, 1:1 지원

자세한 내용은 요금제 페이지를 참고하세요.

Q. 결제는 어떻게 이루어지나요?

+

Toss Payments를 통해 안전하게 결제됩니다:

  • 신용카드 자동결제 (빌링키 방식)
  • 매월 자동 갱신
  • 언제든지 취소 가능

Q. 환불 정책은 어떻게 되나요?

+

환불 정책은 다음과 같습니다:

  • 서비스 이용 전: 전액 환불
  • 7일 이내: 사용일 제외 일할 계산 환불
  • 7일 경과 후: 환불 불가 (단, 서비스 장애 시 협의)

자세한 내용은 환불정책 페이지를 참고하세요.

Q. 구독을 취소하면 즉시 서비스가 중단되나요?

+

아닙니다. 구독 취소 시 다음 결제일까지 서비스를 계속 이용할 수 있습니다.

예: 12월 1일 결제 후 12월 15일 취소 → 12월 31일까지 이용 가능, 1월 1일 자동 갱신 안 됨

Q. 자동매매는 어떻게 작동하나요?

+

코인펄스는 기술적 지표를 활용한 자동매매 보조 시스템입니다:

  1. 사용자가 설정한 매수/매도 조건 모니터링
  2. 조건 충족 시 업비트 API로 주문 실행
  3. 실시간 알림으로 거래 내역 통보
  4. 대시보드에서 수익률 및 통계 확인
💡 지원 지표: SMA, RSI, MACD, 볼린저밴드, SuperTrend

Q. 자동매매로 수익을 보장받을 수 있나요?

+
⚠️ 중요: 코인펄스는 자동매매 보조 도구이며, 수익을 보장하지 않습니다.

암호화폐 거래는 고위험 투자이며 원금 손실 가능성이 있습니다. 모든 투자 결정과 책임은 본인에게 있습니다.

Q. 급등 예측의 정확도는 얼마나 되나요?

+

2024년 백테스트 결과:

  • 전체 정확도: 81.25%
  • 급등 감지율: 75%
  • 평균 선행 시간: 4~6시간
💡 참고: 과거 성과가 미래 수익을 보장하지 않으며, 급등 예측은 참고 자료로만 활용하세요.

Q. 자동매매를 언제든지 중지할 수 있나요?

+

네, 대시보드에서 언제든지 자동매매를 중지할 수 있습니다.

중지 후에는 새로운 주문이 실행되지 않으며, 이미 체결된 주문은 그대로 유지됩니다.

Q. 업비트 API 키가 안전한가요?

+

코인펄스는 API 키를 안전하게 보호합니다:

  • AES-256 암호화: 업계 표준 암호화 방식
  • 출금 권한 미사용: 읽기 및 거래 권한만 사용
  • 접근 제한: 암호화된 형태로만 저장
  • 정기 보안 점검: 월 1회 보안 감사

Q. 출금이 가능한가요?

+
❌ 아니오! 코인펄스는 출금 권한을 절대 요구하지 않으며, 사용하지도 않습니다.

사용자의 자산은 업비트 계정에 안전하게 보관되며, 출금은 업비트에서만 가능합니다.

Q. 개인정보는 어떻게 보호되나요?

+

개인정보 보호를 위한 조치:

  • 최소한의 정보만 수집 (이메일, 비밀번호)
  • 비밀번호는 bcrypt 해싱으로 암호화
  • HTTPS 통신으로 데이터 전송
  • 제3자 제공 없음 (법령 요구 시 제외)

자세한 내용은 개인정보처리방침을 참고하세요.

Q. 모바일에서도 사용할 수 있나요?

+

네, 코인펄스는 모바일 브라우저에서도 사용 가능합니다.

현재 반응형 웹으로 제공되며, 향후 전용 모바일 앱 출시를 계획 중입니다.

Q. 다른 거래소도 지원하나요?

+

현재는 업비트만 지원합니다.

향후 빗썸, 코인원 등 국내 주요 거래소 지원을 계획 중이며, 해외 거래소(바이낸스, 바이비트 등)도 검토 중입니다.

Q. 베타 테스트 참여 혜택이 있나요?

+

베타 테스트 참여자에게는 다음 혜택을 제공합니다:

  • 정식 출시 후 평생 할인 (20% 할인)
  • 신기능 우선 체험
  • 피드백 반영 시 추가 혜택
💡 참여 방법: 지금 가입하시면 자동으로 베타 테스터로 등록됩니다!
